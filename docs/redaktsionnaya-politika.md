# Редакционная политика (контент сайта)

## Пунктуация

- **Точку с запятой (`;`) не используем** — ни в абзацах, ни в перечислениях внутри предложения. Заменяем точкой, двоеточием, тире или делим на два предложения.

## Названия брендов

- **Dodo Pizza** в русском тексте всегда пишем **«Додо Пицца»** (кириллица, кавычки по контексту).

## URL и файлы кейсов

- Для бренда **Яндекс** в именах файлов и сегментах URL используем **`yandex`**, не транслит **`yandeks`**.
