Контекст

Сеть пилотировала WFM-решение для директора магазина. Ручное планирование и алгоритмическое расписание по-разному ощущаются директорами и линейным персоналом — без исследования опыта легко закрепить в продукте решения, которые увеличивают формальную эффективность и одновременно ломают работу магазина.

Запрос

Провести EX-исследование опыта директора и сотрудников розницы при планировании времени: сравнить «вручную» и с использованием WFM; найти способы улучшения опыта пользователей системы.

Процесс

Анализ данных. Изучение текущей архитектуры решения, имеющихся отзывов.

Исследованиех. Интервью и наблюдения с директорами и персоналом магазинов.

Аналитика. Рабочие встречи с командой заказчика для уточнения инсайтов.

Рекомендации. Выработка рекомендаций по улучшению взаимодействия с WFM-cистеммой.

Результат

Employee User Journey Map планирования и управления временем (с WFM и без); пул проблематик; перечень плюсов пилотной версии WFM; рекомендации по доработке системы; список инициатив по внедрению (совместно с заказчиком).
