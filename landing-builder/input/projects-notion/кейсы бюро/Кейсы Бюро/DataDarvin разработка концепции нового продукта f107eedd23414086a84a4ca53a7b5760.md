# DataDarvin: разработка концепции нового продукта

Год выполнения: 2019
Чей опыт: Клиентов
Отрасль: IT, Маркетинг
Тип: Концепция
Прогресс: Завершен
В КП: No

## Ситуация

ИТ-Компания Pulse WiFi обратилась в Бюро для радикального переосмысления своего продукта. Рынок wifi-маркетинга уже формируется, на нем есть игроки. Необходимо было предложить радикально новый смысл для продукта, но сохранить технологию, лежащую в основе продукта.

## Задача, которую решали

- Разработать концепцию нового продукта
